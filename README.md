# Truco Game

Protótipo de um jogo de truco com `React`, `TypeScript`, `Three.js` e base preparada para empacotamento mobile com `Capacitor`.

## Visão do projeto

O objetivo é evoluir este protótipo para um jogo single-player de truco com:

- jogador humano + parceira IA
- dupla adversária controlada por IA
- progressão por campeonatos
- ambientação que evolui de botecos de periferia até um torneio mundial
- jogo offline com ranking online

## Stack atual

- `React`
- `TypeScript`
- `Vite`
- `Three.js`
- `Capacitor`

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```
